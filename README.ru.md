# AmneziaWG FreeBSD Setup + Split Tunneling

Этот репозиторий содержит скрипт для автоматизированной сборки и настройки **AmneziaWG** на **FreeBSD 14/15**. Скрипт позволяет развернуть VPN с защитой от DPI, используя нативный модуль ядра для максимальной производительности.

---

## Возможности

* **Нативная производительность**: Сборка модуля ядра (`if_wg.ko`) специально под вашу версию FreeBSD.
* **Умный патчинг**: Модификация `awg-quick` для работы с системным `ifconfig` без лишних зависимостей.
* **Раздельное туннелирование (Опционально)**: Настройка маршрутизации так, чтобы через VPN шел трафик только для выбранных доменов.
* **Автозапуск при загрузке**: Регистрация сервиса RC (`/usr/local/etc/rc.d/amneziawg`), который автоматически поднимает туннель после перезагрузки системы.
* **Полное удаление**: Специальный флаг `-u` для безопасной очистки системы от всех установленных компонентов.

---

## Системные требования

1. **ОС**: FreeBSD 14.0 или новее (поддерживается FreeBSD 15-CURRENT).
2. **Права**: Запуск от имени **root** (или через `sudo`).
3. **Исходники ядра**: Необходимы для компиляции модуля. Если их нет:
   `freebsd-update fetch install`
4. **Конфигурация**: Необходим рабочий `.conf` файл от вашего провайдера AmneziaWG.

## Установка и использование

### 1. Базовая установка
Для обычной установки AmneziaWG просто укажите путь к вашему конфигурационному файлу:
`sudo ./awg-setup.sh -c /путь/к/vpn.conf`

### 2. Продвинутая установка (Split Tunneling)
Если вы хотите, чтобы через VPN шел трафик только к определенным ресурсам, используйте опцию `-d`:
`sudo ./awg-setup.sh -c /путь/к/vpn.conf -d "rutracker.org,nnmclub.to"`

### Все доступные опции
* `-c FILE`: **(Обязательно)** Путь к вашему `.conf` файлу AmneziaWG.
* `-d DOMAINS`: Опциональный список доменов через запятую для туннелирования. (По умолчанию: `rutracker.org`).
* `-i IFACE`: Имя интерфейса туннеля. (По умолчанию: `awg0`).
* `-u`: Полное удаление и очистка системы.

## Управление сервисом

Туннель остается активным после перезагрузки благодаря интеграции с системой RC.

| Действие | Команда |
| :--- | :--- |
| **Запустить VPN** | service amneziawg start |
| **Остановить VPN** | service amneziawg stop |
| **Статус** | service amneziawg status |
| **Инфо об интерфейсе** | awg show awg0 |

---

## Отладка и логи

* **Лог установки**: Подробности процесса сборки: `/var/log/awg-setup.log`.
* **Работа маршрутов**: События раздельного туннелирования пишутся в системный журнал:
  `grep awg-split /var/log/messages`
* **Ошибка ядра**: Убедитесь, что исходники в `/usr/src` соответствуют версии вашего ядра (`uname -r`).
